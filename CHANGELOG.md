# Changelog

All notable, user-facing changes to open-subtitle are documented here. The
format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and this
project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## 0.0.1

Initial public line of the open-subtitle engine:

- Keyless subtitle engine and CLI (`ost`) / daemon (`ostd`)
- Multi-provider search, library scan, wanted list, and scheduled re-search
- OpenSubtitles.org multi-language search improvements
- GitHub Release binaries and Nix/Cachix packaging
- Workspace crates are not published to crates.io on this line
